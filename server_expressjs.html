CORS on Express.js

In your Express.js app on Node.js:

Install the CORS middleware from NPM

Then use it before any route that depends on it.

const cors = require('cors');
app.use(cors({
  origin: "YOUR-DOMAIN.TLD",
  allowedHeaders: ["Origin", "X-Requested-With", "Content-Type", "Accept"]
}));

app.get('/', function(req, res, next) {
  // Handle the get for this route
});

app.post('/', function(req, res, next) {
 // Handle the post for this route
});

Do read the documentation to learn how to tailor it for your specific needs.
